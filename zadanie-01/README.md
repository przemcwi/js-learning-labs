Zadanie 01 — Liczenie wystąpień elementów w tablicy
Cel

Policzyć ile razy każdy element występuje w tablicy, używając:

zwykłego obiektu {},

oraz Map.

Zadania
1. countOccurrencesObj(array)

Zwraca obiekt, gdzie klucze to elementy tablicy, a wartości to liczba wystąpień.

2. countOccurrencesMap(array)

Zwraca Map, w której klucze to elementy tablicy, a wartości to liczba wystąpień.
